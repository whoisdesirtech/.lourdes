## 4. Logo Usage Guidelines

### Observed Variations
1. **White-on-Red Script Logo**: Clean white SVG vector logo positioned on solid red backgrounds (packaging & header overlays).
2. **Red-on-White Script Logo**: Red Spencerian wordmark on neutral/white cards.
3. **Circular Trademark / Bottle Badge**: A circular graphic containing the contour bottle silhouette combined with the script wordmark.

### Clear Space Rules
* Minimum clear space boundary of `0.5x` (where `x` is the height of the wordmark text) must block out any incoming copy, banners, or imagery borders.

### Sizing and Proportions
* Digital logo minimum width: `80px`.
* The logo vector must maintain its locked-in horizontal aspect ratio and must never be squished or compressed.

### Do's and Don'ts
* **DO**: Align the script logo horizontally.
* **DO**: Maintain high contrast using brand red or brand white.
* **DON'T**: Re-color the script wordmark in non-brand color codes (e.g., green, purple).
* **DON'T**: Apply gradients, metallic textures, or heavy drop shadows to the digital SVG assets.

## 7. Inconsistency Report
* **Corporate vs. Consumer Color Inconsistency**:
  * Corporate website (`coca-colacompany.com`) references corporate red `#F40009` (Pantone 185 C).
  * Consumer website (`cocacola.com`) references digital design token red `#EA0000` (Pantone 485 C).
  * **Standardization Recommendation**: Standardize consumer-facing web apps and digital UI designs on `#EA0000` for higher vibrance and better accessibility contrast.

---

## 8. Gap Analysis

* **Inferred Element**: Font Fallback Configuration
  * *Rationale*: Since `TCCC-UnityText` is a licensed, proprietary typeface, third-party developers require open fallbacks.
  * *Action Item for User*: Formally approve `Montserrat` or `Outfit` (via Google Fonts) as fallback fonts for systems where `TCCC-UnityText` is unlicensed.
